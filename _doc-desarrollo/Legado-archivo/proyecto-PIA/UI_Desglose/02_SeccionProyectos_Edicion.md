# Lectura rápida de la vista

* **Sección: PAI abierto**
  Representa la unidad de trabajo concreta sobre un inmueble.

* **Cabecera del PAI**
  Sitúa al usuario y concentra control inmediato.

* **Banda de datos básicos del inmueble**
  Permite verificar rápidamente qué activo se está revisando.

* **Banda de resumen ejecutivo**
  Aporta una primera lectura analítica antes del detalle completo.

* **Sección de resultados del análisis**
  Reúne la lectura estructurada del contenido generado.

* **Sección de notas e historial**
  Conserva el rastro humano y operativo del proyecto.
