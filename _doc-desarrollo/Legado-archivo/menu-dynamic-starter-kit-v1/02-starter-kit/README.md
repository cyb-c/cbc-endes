# Starter Kit de Código - Menú Dinámico v1

Este directorio contiene el código de ejemplo para implementar el patrón de menú dinámico v1.

## Estructura

```
02-starter-kit/
├── README.md (este archivo)
├── database.md      # Esquema SQL
├── backend.md       # Handler del endpoint
├── frontend.md      # Hooks y componentes React
└── i18n.md         # Sistema de textos (es-ES)
```

## Uso

Cada archivo contiene el código de ejemplo listo para copiar y adaptar a tu proyecto.

## Notas

- El código de backend está en TypeScript para Cloudflare Workers
- El código de frontend está en React, pero es adaptable a otros frameworks
- El sistema de i18n es simple (solo es-ES en v1)
- No incluye roles, permisos, cache ni configuración dinámica de campos (v1 simplificada)
