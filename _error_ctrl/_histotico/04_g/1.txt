### 2) Estás usando salida estructurada y el SDK no te la deja en `output_text`

Si además estás usando `text.format` para JSON estructurado, la respuesta puede venir en forma estructurada dentro de `output`, no necesariamente como string plano agregado en `output_text`. ## Qué corregir

### Opción A: extraer texto recorriendo `response.output`

Haz fallback a `output` cuando `output_text` venga vacío.

Ejemplo robusto en Node.js:

```js
function extractResponseText(resp) {
  if (resp.output_text && resp.output_text.trim()) {
    return resp.output_text;
  }

  const parts = [];

  for (const item of resp.output || []) {
    if (item.type === "message" && Array.isArray(item.content)) {
      for (const content of item.content) {
        if (content.type === "output_text" && typeof content.text === "string") {
          parts.push(content.text);
        }

        // por si alguna variante del SDK serializa distinto
        if (typeof content.text === "string" && !content.type) {
          parts.push(content.text);
        }

        // algunas salidas estructuradas pueden venir en otros campos
        if (content.json) {
          parts.push(JSON.stringify(content.json));
        }
      }
    }
  }

  return parts.join("\n").trim();
}
```
