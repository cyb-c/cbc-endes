# Lectura rápida de esta vista de pájaro

* La app se apoya en una estructura estable de **top bar + menú lateral + área principal + footer**.
* El núcleo del trabajo vive en el **área principal derecha**.
* Dentro de esa área, la sección central del sistema es **Proyectos PAI**.
* Y dentro de Proyectos, la pantalla más importante es el **PAI abierto**.
