## Uso en componentes

```typescript

// En tu componente
const label = MENU_TEXTS[mod.nombre_mostrar] || mod.nombre_mostrar
```

## Extensión para otros idiomas

Para añadir soporte para más idiomas en v2, crea archivos adicionales:

```javascript
// i18n/en-EN.js
export const MENU_TEXTS = {
  'menu.modulos.panel': 'Panel',
  'menu.modulos.facturacion': 'Billing',
  // ...
}

// i18n/fr-FR.js
export const MENU_TEXTS = {
  'menu.modulos.panel': 'Panneau',
  'menu.modulos.facturacion': 'Facturation',
  // ...
}
```
